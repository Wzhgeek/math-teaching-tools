# 快速开始指南

## macOS 上打包 Windows exe - 三步搞定！

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 测试运行（可选）
```bash
npm start
```

### 3️⃣ 打包 Windows exe
```bash
npm run build-win
```

打包完成后，在 `dist` 文件夹中找到 `.exe` 文件，这就是 Windows 安装程序！

---

## 详细说明

### 系统要求
- ✅ macOS（任何版本）
- ✅ Node.js（LTS 版本推荐）
- ✅ 网络连接（用于下载构建工具）

### 打包命令说明

| 命令 | 说明 |
|------|------|
| `npm run build-win` | 打包 64 位 Windows 应用（推荐） |
| `npm run build-win-ia32` | 打包 32 位 Windows 应用 |
| `npm start` | 本地测试运行 |

### 打包时间
- **首次打包**：5-10 分钟（需要下载构建工具）
- **后续打包**：1-3 分钟

### 输出文件
打包完成后，在 `dist` 文件夹中会生成：
- `数学教学工具 Setup 1.0.0.exe` - Windows 安装程序
- 可以直接复制到 Windows 电脑上安装使用

### 网络问题解决

如果下载速度慢或失败，可以设置国内镜像：

```bash
# 设置 npm 镜像
npm config set registry https://registry.npmmirror.com

# 设置 Electron 镜像（可选）
export ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
```

### 验证打包结果

打包成功后，你会看到类似这样的输出：
```
✔ building        target=NSIS platform=win arch=x64
✔ default         file=dist/数学教学工具 Setup 1.0.0.exe
```

---

## 常见错误解决

### 错误：网络连接超时
**解决**：使用国内镜像源（见上方）

### 错误：权限不足
**解决**：确保有写入 `dist` 文件夹的权限

### 错误：找不到 electron
**解决**：重新运行 `npm install`

---

## 下一步

打包完成后，将 `dist` 文件夹中的 `.exe` 文件复制到 Windows 电脑上，双击安装即可使用！
