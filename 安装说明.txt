==========================================
数学教学工具 - Windows exe 打包说明
==========================================

✅ 重要提示：可以在 macOS 上直接打包 Windows exe 文件！
Electron Builder 支持跨平台打包，无需 Windows 系统。

## 第一步：安装 Node.js
如果还没有安装 Node.js，请先访问 https://nodejs.org/ 下载并安装
（推荐安装 LTS 版本）

## 第二步：安装项目依赖
在项目目录下打开终端（Terminal），运行：

    npm install

这会安装 Electron 和 electron-builder 等必要的依赖包
（首次安装可能需要几分钟时间）

## 第三步：测试运行（可选）
在打包之前，可以先测试一下应用是否正常：

    npm start

如果一切正常，会打开一个窗口显示主菜单。

## 第四步：打包成 Windows exe

### 在 macOS 上打包 Windows 应用：

运行以下命令进行打包：

    npm run build-win

这个命令会在 macOS 上自动下载 Windows 构建工具并打包出 Windows exe 文件。

打包过程可能需要几分钟时间（首次打包会更久，因为需要下载 Windows 构建工具），
完成后会在项目目录下生成一个 `dist` 文件夹，里面包含：
- Windows 安装程序（.exe 文件，文件名类似：数学教学工具 Setup 1.0.0.exe）
- 可以直接在 Windows 系统上运行

### 其他打包选项：

- 打包 32 位 Windows 应用：npm run build-win-ia32
- 打包所有平台：npm run build-all

## 注意事项：
1. ✅ 在 macOS 上可以完美打包 Windows exe，无需 Windows 系统
2. 打包后的 exe 文件会比较大（约 100-200MB），因为包含了 Chromium 浏览器内核
3. 打包时会自动下载 Windows 构建工具和 Electron，需要网络连接
4. 首次打包可能需要较长时间（5-10分钟），因为需要下载大量文件
5. 打包后的 exe 文件可以直接在 Windows 系统上运行，无需安装 Node.js

## 常见问题：

Q: 在 macOS 上能打包 Windows exe 吗？
A: ✅ 完全可以！Electron Builder 支持跨平台打包，会自动下载所需的 Windows 构建工具。

Q: 打包失败怎么办？
A: 确保网络连接正常，Electron 需要从网络下载。如果失败，可以尝试：
   - 使用国内镜像：npm config set registry https://registry.npmmirror.com
   - 检查网络连接，确保能访问 GitHub 和 npm 服务器
   - 如果网络较慢，可以设置代理或使用镜像源

Q: 打包后的文件在哪里？
A: 在项目目录下的 `dist` 文件夹中，文件名类似：数学教学工具 Setup 1.0.0.exe

Q: 打包后的 exe 文件可以直接运行吗？
A: 是的！打包后生成的是 Windows 安装程序，在 Windows 系统上双击安装即可使用。

Q: 首次打包很慢怎么办？
A: 这是正常的，因为需要下载 Windows 构建工具（约 200-300MB）。后续打包会快很多。

Q: 打包时提示找不到 icon.ico？
A: 不用担心，没有图标文件也能正常打包，只是应用没有自定义图标而已。
